What is USLang?

USLang is a revolutionary programming language designed specifically for absolute beginners. Instead of cryptic syntax, USLang uses English-like commands that anyone can understand and write. Perfect for learning programming fundamentals without the intimidation factor!

✨ Features

English-like Syntax

Write code that reads like English sentences

Simple & Clean

No confusing brackets or semicolons

Powerful

Variables, functions, loops, and conditions

Learn Fast

Perfect for beginners of all ages

📦 Installation & Setup

The Easy Way (Recommended)

Just 3 simple steps to get started!

  1. Download - Click the download button above to get USLang-v1.0-Complete.zip
  2. Extract - Right-click the zip file and select "Extract All". Choose any location.
  3. Install - Open the extracted folder and double-click standalone-installer.bat

⚡ After Installation

Close and reopen PowerShell, then run:

us "examples\hello.us"
us "examples\arithmetic.us"
us

The Advanced Way (Build from Source)

If you want to build USLang yourself:

Prerequisites: Rust must be installed. Download from https://www.rust-lang.org/tools/install

cd c:\allpytonfiles\a\uslang
cargo build --release --target x86_64-pc-windows-gnu
.\standalone-installer.bat

Then restart PowerShell and start using USLang from anywhere!

📚 Code Examples

Here are some simple USLang programs to get you started:

Hello World

Say "Hello". Say "World".

Variables

Let name be text "Alice". Let age be number 25. Say name. Say age.

Arithmetic

Let x be number 5 plus 3. Let y be number 10 minus 2. Say x. Say y.

Conditionals

Let age be number 25. If age greater than 18 then. Say "You are an adult". End if.

Loops

Let count be number 1. Loop until count greater than 5. Say count. Let count be count plus 1. End loop.

Functions

Create function greet with name. Say "Hello". Say name. End function. Use function greet with "Bob".

⌨️ Command Reference

Here are the common commands you'll use:

Say "text". - Print text to screen
Let x be number 5. - Create a number variable
Let x be text "hello". - Create a text variable
If condition then ... End if. - Conditional statement
Loop until condition ... End loop. - Loop until condition
Create function name with params ... End function. - Define function
Use function name with params. - Call function
plus, minus, times, divided by - Arithmetic operators
equals, greater than, less than - Comparison operators

❓ Troubleshooting

Command not found

If "us" is not recognized, open a new PowerShell window. The system PATH update takes effect after reopening.

Build failed

Ensure you have Rust installed and the GNU toolchain:

rustup target add x86_64-pc-windows-gnu

Permission denied during install

Right-click install.bat and select "Run as administrator".
